Detailed Description

template<typename TComponent>
class Pool< TComponent >

Template class for storing and managing components of type TComponent.

Template Parameters
TComponentThe type of component to be stored in the pool

This class provides a type-safe container for storing and accessing components. It manages a dynamic array of components with methods for adding, accessing, and modifying elements.
