Detailed Description

Represents the system that manages collision detection and response between entities.

The CollisionSystem checks for circular collisions between entities with CircleColliderComponent and triggers collision events if necessary.

void CollisionSystem::Update ( sol::state & lua)

Updates the collision system to check for collisions between entities.

Parameters
luaA reference to the Lua state used for scripting.

This function iterates through all entities in the system, checking for circular collisions between pairs of entities. If a collision is detected, it triggers the onCollision scripts associated with the entities involved.
