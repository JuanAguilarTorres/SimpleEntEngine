\doxysubsection{Detailed Description}
Works managing pools of components in an entity-\/component system. 

The \doxylink{class_pool}{Pool} system provides efficient storage and access to components in an entity component system architecture. It manages dynamic arrays of components with type safety and convenient access methods.
