\doxysubsection{Detailed Description}
Manages keyboard and mouse input actions. 

The \doxylink{class_controller_manager}{Controller\+Manager} class handles mapping of keyboard keys and mouse buttons to specific actions, tracking their states (pressed or released), and storing the current mouse position.
