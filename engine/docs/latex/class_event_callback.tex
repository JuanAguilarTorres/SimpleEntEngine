\subsubsection*{template$<$typename TOwner, typename TEvent$>$\newline
class Event\+Callback$<$ TOwner, TEvent $>$}
Template class for handling event callbacks. 

The \doxylink{class_event_callback}{Event\+Callback} class allows the binding of a member function of an owner class to an event, so that the member function will be called when the event occurs.
