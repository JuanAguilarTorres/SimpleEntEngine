\doxysubsection{Detailed Description}
Represents the system that manages the movement of entities. 

The \doxylink{class_movement_system}{Movement\+System} updates the position of entities based on their velocity as defined in the \doxylink{struct_rigid_body_component}{Rigid\+Body\+Component}.
