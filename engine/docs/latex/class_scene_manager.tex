\doxysubsection{Detailed Description}
Manages scene lifecycle and transitions in the game engine. 

The \doxylink{class_scene_manager}{Scene\+Manager} class provides high-\/level scene management functionality, including scene loading, transitions, and state management. It works with Lua scripts to define and load scenes, and maintains the current scene state.
