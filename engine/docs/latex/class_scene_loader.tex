\doxysubsection{Detailed Description}
Manages loading and initialization of game scenes from Lua scripts. 

The \doxylink{class_scene_loader}{Scene\+Loader} class is responsible for parsing scene files written in Lua and creating the corresponding game objects, assets, and configurations. It handles loading of various assets including videos, sprites, music, sound effects, and fonts, as well as setting up input controls and entities.
