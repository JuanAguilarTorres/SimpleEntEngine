\doxysubsection{Detailed Description}
A system responsible for displaying hitboxes in the game. 

This system is responsible for rendering the hitboxes of entities that have a {\ttfamily \doxylink{struct_box_collider_component}{Box\+Collider\+Component}} and a {\ttfamily \doxylink{struct_transform_component}{Transform\+Component}}. It is typically used for debugging purposes to visualize collision boundaries.
