\doxysubsection{Detailed Description}
A system responsible for detecting and handling overlaps between entities. 

The {\ttfamily \doxylink{class_overlap_system}{Overlap\+System}} is designed to check collisions between entities and handle the overlap behavior based on their positions and components.
