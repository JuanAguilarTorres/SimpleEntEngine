\doxysubsection{Detailed Description}
Asset Management system for handling game resources. 

The \doxylink{class_asset_manager}{Asset\+Manager} class provides a centralized system for loading, storing, and managing different types of game assets including\+:
\begin{DoxyItemize}
\item Textures (SDL\+\_\+\+Texture)
\item Fonts (TTF\+\_\+\+Font)
\item Sound Effects (Mix\+\_\+\+Chunk)
\item Music Tracks (Mix\+\_\+\+Chunk)
\item Video Files (using FFmpeg)
\end{DoxyItemize}

All assets are stored in memory and can be accessed using their unique identifiers. The manager handles proper cleanup of resources when destroyed.

\begin{DoxyNote}{Note}
This class requires SDL2, SDL2\+\_\+ttf, SDL2\+\_\+mixer, and FFmpeg libraries 
\end{DoxyNote}
