\doxysubsection{Detailed Description}
Represents the system that manages box collisions between entities. 

The \doxylink{class_box_collision_system}{Box\+Collision\+System} detects and manages collisions between entities with box colliders. It checks for Axis-\/\+Aligned Bounding Box (AABB) collisions and triggers on\+Collision scripts when collisions occur.
