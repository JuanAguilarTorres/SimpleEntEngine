\doxysubsection{Detailed Description}
Represents the system that manages collision detection and response between entities. 

The \doxylink{class_collision_system}{Collision\+System} checks for circular collisions between entities with \doxylink{struct_circle_collider_component}{Circle\+Collider\+Component} and triggers collision events if necessary.
