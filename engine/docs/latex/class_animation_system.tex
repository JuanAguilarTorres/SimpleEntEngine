\doxysubsection{Detailed Description}
Represents the system that manages and updates animations for entities. 

The \doxylink{class_animation_system}{Animation\+System} processes all entities with \doxylink{struct_animation_component}{Animation\+Component} and \doxylink{struct_sprite_component}{Sprite\+Component}, updating their current animation frame based on elapsed time and frame speed rate.
